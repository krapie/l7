# l7

l7 is a layer 7 load balancer built from scratch in Go.

## Installation

```bash
# Clone the repository
git clone https://github.com/krapie/l7.git

# Build the binary
cd l7
make build
```

## Usage

```bash
# Start the load balancer
./bin/l7
```

For more usage, see [USAGE.md](./USAGE.md).
